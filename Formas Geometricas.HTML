Como Criar Formas Geométricas

Círculo

Para criar um círculo, você precisa de um ponto central (o centro) e um raio que define o tamanho do círculo.

Área do Círculo: A = π * r²

Perímetro do Círculo: P = 2 * π * r

Triângulo

Para criar um triângulo, você precisa de três lados. Se conhecemos a base e a altura, podemos calcular a área.

Área do Triângulo: A = (base * altura) / 2

Perímetro do Triângulo: P = lado1 + lado2 + lado3

Quadrado

Para criar um quadrado, você precisa de quatro lados de igual comprimento.

Área do Quadrado: A = lado * lado

Perímetro do Quadrado: P = 4 * lado

Retângulo

Para criar um retângulo, você precisa de dois pares de lados opostos com tamanhos iguais.

Área do Retângulo: A = comprimento * largura

Perímetro do Retângulo: P = 2 * (comprimento + largura)
