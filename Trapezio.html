Qual as características de um Trapezio?

Definimos como trapézio todo quadrilátero que possui dois lados paralelos. Os lados paralelos são conhecidos como

base maior e base menor. Como todo quadrilátero, possui duas diagonais, e a soma dos ângulos internos é igual a 360º.
